Set up your workspace

Like other Carpentries workshops, you will be learning by "coding along" with the Instructors. To do this, you will need to have both the window for the tool you will be learning about (a terminal, RStudio, your web browser, etc..) and the window for the Zoom video conference client open. Plus, for parts of the workshop, your graphical user interface (Windows Explorer on PC, Finder on MacOS). In order to see them at once, we recommend using one of the following set up options:

  • Two monitors: If you have more than one monitor, plan to have the tool you are learning (i.e. the shell) on one monitor, the video conferencing software on another, and your graphical user interface on yet another.
  • Two devices: If you don't have multiple monitors, do you have another device (tablet, smartphone) with a medium to large sized screen? If so, try using the smaller device as your video conference connection and your larger device (laptop or desktop) to follow along with the tool you will be learning about.
  • Divide your screen: If you only have one device and one screen, practice having three windows (the Zoom window, the shell and your computer's graphical user interface (GUI)) open together. How can you best fit them on your screen? Will it work better for you to open them in separate windows, toggling between them using a keyboard shortcut such as Alt-TAB? Try it out in advance to decide what will work best for you.
